# Installations

## Typescript

```shell
yarn add -D typescript
```

## Express

```shell
yarn add express
yarn add -D @types/express @types/body-parser
```

## CORS

```shell
yarn add cors
yarn add -D @types/cors
```

## Config

```shell
yarn add config
yarn add -D @types/config
```
